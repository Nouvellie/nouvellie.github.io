PROJECTS

Thesis: Nouvellie Social Network

Creation of a social network model as a channel of diffusion, grouping and musical communication, developed in Python Language with the Django Framework, and a PostgreSQL as database motor. The main idea of this social network is to help people in the creation of bands and, what is more important, in their transparent and massive diffusion. It works with profiles of people associated to an instrument and musical style, to filter in searches quickly and to be able to create bands in a more efficient and personalized way, and at the same time to be able to spread music and information openly and without needing to spend money to achieve it.

Geometric Meshes: Algorithm

Improvement of the quality of the 2D geometric meshes of the Delaunay triangles, by means of criteria in the limit of the angles, refining Africa of 9665 triangles. Developed in Python with the Numpy library, sequential and parallel with MPI, a hybrid version of Rivara's 4T method. Programmed from scratch, creating all the mathematical area, and operation. It started as a project in c and c++, which became quite complicated, to then continue with Java, and finally we decided to do everything from scratch keeping the logic using and learning Python at the same time, given the ease provided by this programming language with its libraries. The project was at the level of our teacher's thesis in the course of Parallel Computing (tenth semester).

Fia-2s: Software

Construction of software in Visual Studio with Visual Basic, adapting a payment system, reliable purchase (money back guarantee), balances and inventories to a small company. All managed on the mysql server.
